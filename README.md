# Pantry Tracker - Legal Documents

Host these files on GitHub Pages for App Store / Play Store submission.

## Setup

1. Create new repo: `pantrytracker-legal`
2. Copy these files
3. Go to repo Settings → Pages → Enable from `main` branch
4. Your URLs will be:
   - `https://YOURUSERNAME.github.io/pantrytracker-legal/`
   - `https://YOURUSERNAME.github.io/pantrytracker-legal/privacy`
   - `https://YOURUSERNAME.github.io/pantrytracker-legal/terms`
